iMac Retina 27 as the External Display

2016-07-03

It is known that the iMac Retina 27 inch does not support the Target Display Mode. And you cannot use an iMac as an external display. However, there is a workaround---use thunderblot bridge and screen sharing. Here is the setup.

  1. Connect your Macbook with iMac.
  2. In the network, you should see a thunder bridge connection on both your iMac and Macbook. We only care the connection in your Macbook.
  3. @Macbook, assign an manually IP address to your thunder bridge as shown below. (such that you won't need to reconnect with another IP address when you disconnected from the iMac)
  4. @iMac, use screen sharing application to connect to your Macbook.

Thunder Bridge Configuration

I can still feel latency in the screen sharing. But it is workable:-)

Now, one technical question--why should we assign an address in the form of 169.254.x.x? This kind address is a link-local address, which is valid only for communications within the network segment or the broadcast domain that the host is connected to.

In RFC 3927, the Internet Engineering Task Force has reserved the address block 169.254.0.0/16. From this block, the address range 169.254.1.0 to 169.254.254.255 may be used for link-local addressing in Internet Protocol Version 4.

Microsoft refers to this address autoconfiguration method as Automatic Private IP Addressing (APIPA). It is sometimes also casually referred to as auto-IP.
