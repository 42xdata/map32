HyperSwitch: Switch back to active window any fullscreen window.

2016-06-13

  • Problem: you can not swich back to an full screen instance or active window in OS X using cmd + tab.
  • Solution: use HyperSwitch.
  • Details.

Until today, I found that when I try to switch full screen instances of applications with cmd + tab in OS X, OS X does not switch the full screen window.

Someone gives a solution at stackexchange.

The short answer is to use HyperSwitch, a better keyboard window switcher for Mac. It is currently in beta but free.

You can customized your key there. For we, I just use it to override the default cmd+tab.

A bouns point: you can also switch back to the active window! I found that I quite often get into such a scenrio. Never know that it can be solved by some application!
