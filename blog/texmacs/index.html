TeXmacs is a Great Tool. I've heard good words about TeXmacs (written by Joris van der Hoeven) for a long time. But since it does not support retina display, I never treat it seriously for daily use.

Finally, Massimiliano Gubinelli in the mailing list shares his private copy which is hacked to support Retinaüéâ

It is stable enough for my daily use (Some crashes happen occasionally.).

Some earlier post (in Chinese 1,2) recommend the software for writing LaTex.

The TeXmacs workshop videos are very comprehensive and easy to understand. If you are impatient, you may go to the 4-th video "the TeXmacs document format, style files, and converters" first. It shows the design philosophy behind the TeXmacs.

I think TeXmacs is really a great piece but undervalued software. Hope it can get better and better.
