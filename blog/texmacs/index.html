Some earlier post (in Chinese 12) recommend the software for writing LaTex.

The TeXmacs workshop videos are very comprehensive and easy to understand. If you are impatient, you may go to the 4-th video "the TeXmacs document format, style files, and converters" first. It shows the design philosophy behind the TeXmacs.

The most useful tips is:

  • Cmd + Option +U: switch to/back the source code mode, so you can edit the internal representation easily.

In all, TeXmacs is really a great piece but undervalued software. Hope it can get better and better.
