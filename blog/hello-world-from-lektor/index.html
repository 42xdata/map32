How to install a plugin?

For example, you may want to install this one

  1. Install the local plugin packages/X
lektor list
  1. Serve locally.
lektor build.

  1. If step <1> fails, make sure the dependencies packages/X/setup.py are listed. e.g. install_requires=['pypandoc', 'bs4']. Fix the mistake and go to step <1> agian.

  2. If you cannot reinstall the plugins in step 3, try to flush cache first.

lektor plugins flush-cache
