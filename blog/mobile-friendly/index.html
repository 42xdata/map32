Mobile Friendly Website

2016-06-15

I realize that my website is not mobile friendly1. I am not an expert in web design, so bear with me. And here is my notes on improve the mobile friendness. The reference will be 2.

The firse improvement is add a viewpoint:

<meta name=viewport content="width=device-width, initial-scale=1">

Now, it is much better. But

The page content is too wide for the viewport, forcing the user to scroll horizontally. Size the page content to the viewport to provide a better user experience. The page content is 779 CSS pixels wide, but the viewport is only 411 CSS pixels wide.
