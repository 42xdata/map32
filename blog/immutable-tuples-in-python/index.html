Immutable Tuples in Python

>>> from collections import namedtuple
>>> P = namedtuple("Point", ['x', 'y'])
>>> a = P(1,2)
>>> a
Point(x=1, y=2)
>>> a.x=3
Traceback (most recent call last):
  File "<input>", line 1, in <module>
    a.x=3
AttributeError: can't set attribute
