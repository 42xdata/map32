Python Staic Class Variable

2016-06-13

Note that the class level varible MUST be refered via the class name.

The following snippet explains the differences:

>>> class MyClass:
...     x = 3
>>> m = MyClass()
>>> m.x = 4
>>> MyClass.i, m.i
>>> (3, 4)
