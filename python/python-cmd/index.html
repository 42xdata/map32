Writing Script Using Python

2013-08-06

There are many old options to call external shell command from python. But the standard way is to use subprocess.

PEP 324 ‚Äì PEP proposing the subprocess module is a good resource to understand the philosophy behind subprocess module.

An Example

  • Input: assignment3/student_name/Assignment.zip
  • Output: extract all Assignment.zip and delete the zip file
raw_args = "find assignment3 | grep .zip"
output = subprocess.check_output(raw_args, shell=True)
files = output.split()
for f in files:
        i = f.rfind('/')
        subprocess.call('unzip %s -d %s'%(f, f[:i]), shell =True)
        subprocess.call('rm %s'%(f), shell=True)

Call()

subprocess.call() aims to be an enhancement over os.system(), while still very easy to use:

The motivation behind the call() function is simple: Starting a process and wait for it to finish is a common task.

Consider this example:

os.system("stty sane -F " + device)

With subprocess.call(), this would look like:

subprocess.call(["stty", "sane", "-F", device])

or, if executing through the shell:

subprocess.call("stty sane -F " + device, shell=True)
