Comprehension Syntax in Python

2016-06-20

Comprehension syntax is a common idiom in Python and mostly known in LINQ. But it is invented much earlier, perhaps you are most familiar with the set comprehension like:

\[S=\{x | p(x) is true \}\]

In Python, you can use comprehension syntax for list, set, dict

>>> {x: x**2 for x in (2, 4, 6)}
{2: 4, 4: 16, 6: 36}
